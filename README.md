# LPサイト模写

## 概要
既存のLP型企業サイトを参考に、HTML/CSS/JavaScriptを用いて模写制作しました。
レスポンシブ対応を行い、スマホでも閲覧しやすいよう調整しています。

## 制作目的
- HTML/CSSでのレイアウト構築に慣れる為
- JavaScript/jQueryを使った動きを実装する為
- 実務でありそうなレスポンシブ対応を意識する為

## 使用技術
- HTML
- CSS
- JavaScript(basic)
- jQuery
- Figma
- Canva

## 担当範囲
- デザインカンプを元にコーディング
- レスポンシブ対応
- スムーススクロール

## 工夫した点
- お借りした素材は使わず、全て自分で用意しました
- 作成するデザインを改めて考え、デザインカンプは作り直しました
- 配布ページのヒントを見ながら、自分なりに答えを出し作成しました

## 苦労した点・解決方法
- レスポンシブデザイン作成に不慣れな為、バランス調整が大変でした
- わからない部分は書籍やネットで検索して解決しました
- それでもわからない場合は、AI(Copilot)を使用して解決しました
- コンタクトページ、ニュースページは自作デザインです(追加)

## 今後改善したい点
- 一部模写し切れてない部分があるので、もっと再現度を高めたいです
