# LPサイト模写

## 概要
デザイン・コーディング練習として制作した、LP型企業サイトの模写作品です。
参考デザインを元に、Figmaでの構成再設計からHTML / CSSによるコーディングまでを行いました。

## 制作目的
HTML / CSS の基礎理解と、レイアウト・余白設計の練習を目的に制作しました。
その他、Figma操作練習も兼ねています。

## 工夫した点
- 出来る限り忠実に元デザインを再現。
- レスポンシブ対応（スマートフォン表示）
- レスポンシブでの画像サイズ変更処理
- 一部画像をCanvaで作成・加工
